# ARM CPU Simulator

A Python-based CPU simulator developed for CP216: Microprocessors at Wilfrid Laurier University.

## Overview

This project simulates core components of a processor, including instruction execution, memory management, and cache behavior. The simulator supports both ARM and THUMB instruction sets and models how instructions are decoded, executed, and stored in memory.

The project was developed to explore fundamental concepts in computer architecture and microprocessor design through a practical software implementation.

## Features

* ARM instruction decoding and execution
* THUMB instruction decoding and execution
* Memory management and address handling
* Cache simulation
* Program execution and benchmarking
* Automated testing utilities

## Technologies

* Python
* Object-Oriented Programming
* Computer Architecture
* Memory Management
* Cache Simulation

## Running the Simulator

```bash
python cpu.py
```

Benchmarking and testing utilities are also included within the project repository.

## Notes

This project was developed as a team project for CP216: Microprocessors at Wilfrid Laurier University and demonstrates concepts related to processor architecture, instruction execution, memory systems, and cache performance.
